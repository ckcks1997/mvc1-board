MVC1 게시판 개인 프로젝트
===================
JSP를 이용한 간단 게시판

## 구현한 기능
* 회원가입, 로그인 
* 게시판 기능(읽기/쓰기/수정/삭제, 수정시 아이디 세션 검증, 조회수 카운트)

## 사용한 기술
* JAVA
* Tomcat 9
* JSP
* BootStrap 5 + Font Awesome
